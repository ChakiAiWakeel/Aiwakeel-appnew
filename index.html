Häufige Fragen
✦
Wie funktioniert das Zero Touch Setup? +
Sie melden sich mit Google an. Der Agent erkennt automatisch Ihre Search Console Domains, Google Ads Konten und Budgets. Fehlende Kampagnen werden automatisch erstellt - ohne manuelle Eingabe.
Wie kündige ich das Abo? +
Jederzeit mit einem Klick über das Stripe-Kundenportal zum Ende des Abrechnungszeitraums kündigen.
Kann ich den Agenten pausieren? +
Ja. Im Kunden-Dashboard können Sie den Agenten jederzeit pausieren oder wieder aktivieren - ohne Kündigung.
Wer zahlt das Google Ads Budget? +
Ihr Werbebudget wird direkt von Google abgebucht. Unsere Gebühr deckt ausschließlich die intelligente Steuerung.
Sind meine Daten sicher? +
Ja. Hosting auf deutschen Servern, DSGVO-konform, nur offizielle Google-APIs - keine Passwörter gespeichert.
Wie funktioniert die WhatsApp-Steuerung? +
Im Premium-Plan stellt der Agent proaktive Updates per WhatsApp zu und nimmt Befehle wie "Budget erhöhen" oder "Kampagne pausieren" direkt entgegen.
